#### Example Quest and Dialog System
Example project for [Quest and Dialog System plugin](https://github.com/mrkriv/QaDS)


#### Install

 - git clone https://github.com/mrkriv/ExampleQaDS.git
 - cd .\ExampleQaDS\
 - git submodule init
 - git submodule update
